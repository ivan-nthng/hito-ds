# Typography System

Our typography system is built with readability and scalability in mind. It uses a modular scale and consistent line heights to maintain vertical rhythm.

## Font Families

### Sans Serif

## Usage

### Headings

```tsx
<h1 className="text-5xl font-bold">Heading 1</h1>
<h2 className="text-4xl font-bold">Heading 2</h2>
<h3 className="text-3xl font-semibold">Heading 3</h3>
<h4 className="text-2xl font-semibold">Heading 4</h4>
```

### Body Text

```tsx
<p className="text-base">Default body text</p>
<p className="text-lg">Larger body text</p>
<p className="text-sm">Smaller body text</p>
```

### Font Weights

```tsx
<p className="font-normal">Normal text</p>
<p className="font-medium">Medium text</p>
<p className="font-bold">Bold text</p>
```

### Line Heights

```tsx
<p className="leading-none">No line height</p>
<p className="leading-normal">Normal line height</p>
<p className="leading-relaxed">Relaxed line height</p>
```
