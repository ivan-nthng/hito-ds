# Color System

Our color system is built with accessibility and flexibility in mind. Colors are defined using HSL values for better manipulation and theme support.

## Usage

### Direct Color Usage

```tsx
<div className="bg-primary-500">Primary Color</div>
<div className="text-gray-700">Gray Text</div>
```

### Theme-Aware Usage

```tsx
<div className="bg-background text-foreground">Theme-aware content</div>
```

### Dark Mode Support

```tsx
<div className="bg-primary-500 dark:bg-primary-600">Dark mode supported content</div>
```
